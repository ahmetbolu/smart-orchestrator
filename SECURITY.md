# Security Policy

## Supported Versions

Currently, only the latest version of the `smart-orchestrator` guidelines is supported with security updates.

## Reporting a Vulnerability

If you discover a vulnerability related to Prompt Injection, AI Jailbreaking, or any structural flaw in the `AGENTS.md` templates that causes the AI to behave maliciously or bypass the Auto-Mechanism, please do NOT open a public issue.

Instead, please send an email or contact the repository maintainers privately. We will evaluate the report and issue a patch or update the recommended prompt structure to mitigate the risk.
