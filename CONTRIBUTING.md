# Contributing to Smart Orchestrator

First off, thanks for taking the time to contribute! This project aims to define the industry standard for AI Auto-Mechanism and Skill Triage.

## How Can I Contribute?

### 1. Reporting Bugs

If you find a flaw in the prompt architecture or an edge case where the AI ignores the triage rule, please submit a bug report using the `.github/ISSUE_TEMPLATE/bug_report.md` template.

### 2. Suggesting Enhancements

Have an idea for a better architectural flow or a new mandatory protocol? Submit a feature request using the `.github/ISSUE_TEMPLATE/feature_request.md` template.

### 3. Adding Templates & Examples

We welcome new examples of `SKILL.md` files or prompt injections for different AI models (e.g., ChatGPT, Claude, GitHub Copilot).

- Add new AI templates to the `templates/` folder.
- Add new skill examples to the `examples/` folder.

## Pull Request Process

1. Fork the repo and create your branch from `main`.
2. Ensure your markdown is well-formatted.
3. Update the README.md with details of changes if applicable.
4. Issue that pull request!

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
